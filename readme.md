# DMs Battlefield

Este projeto é uma aplicação web frontend desenvolvida para simular um campo de batalha interativo, visando **facilitar a organização de combates em RPGs** como Dungeons & Dragons. A aplicação oferece uma interface para visualização e gerenciamento de combates, com foco na usabilidade e responsividade.

## Funcionalidades

-   **Campo de Batalha Interativo**: Permite a visualização e manipulação de elementos em um ambiente digital, otimizando a organização de combates.
-   **PWA (Progressive Web App)**: Suporte para instalação e uso offline, garantindo acesso rápido e desempenho otimizado em diversos dispositivos.

## Tecnologias Utilizadas

O projeto é construído com as seguintes tecnologias:

-   **Frontend Framework**: React
-   **Build Tool**: Vite
-   **Linguagem**: TypeScript
-   **Estilização**: TailwindCSS
-   **Componentes UI**: Radix UI
-   **Funcionalidade Drag and Drop**: dnd-kit
-   **Plataforma de Deploy**: Vercel
